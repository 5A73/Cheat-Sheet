### xrdp
Kali LinuxにXRDPをインストールしてリモートデスクトップ接続を行うための手順を以下に示します。

1. XRDPのインストール
ターミナルを開き、以下のコマンドでXRDPをインストールします：

```bash
sudo apt update
sudo apt install xrdp
```
2. XRDPサービスを開始し、自動起動を有効にします：

```bash
sudo systemctl enable xrdp
sudo systemctl start xrdp
```
